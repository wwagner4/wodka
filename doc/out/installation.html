prerequisites

Wodka is a java application.To run it you need the java virtual machine on your computer. If you have a Mac you are lucky as it is already installed. The others can get it from java.com for free.

You need version 1.4 or higher.

Test it by typing in a commandline window (console) 'java -version'. The result should be something like 'java version 1.5.0_02 Java (TM) 2 RuntimeEnvironment...'

download

Download the the latest wodka release from the wodka download page .The name is wodka vx.x (e.g. wodka v0.22).

One release consists usually of a single zip file that contains executables, libraries and source code.

installation

  • Expand the downloaded zip file anywhere on your computer. E.g. to 'c:\Program Files' or '/usr/local/'
  • Set the JAVA_HOME environment variable to the home directory of your java runtime environment. E.g. JAVA_HOME='C:\Programme\Java\jdk1.5.0_02'.
  • If you run wodka under linux you will have to generate your own script files to start wodka. Take the .bat files in the /bin/ directory as a template.

Widows user may replace the '%JAVA_HOME%/bin/java' strings in the /bin/.bat files by java. In that case you do not have to set the JAVA_HOME environment variable.
